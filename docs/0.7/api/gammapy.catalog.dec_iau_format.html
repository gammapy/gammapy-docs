dec_iau_format¶

gammapy.catalog.dec_iau_format(dec, digits)[source]¶

Declination part of an IAU source designation.

Reference: http://cdsweb.u-strasbg.fr/Dic/iau-spec.html

digits format
2 +DD
3 +DDd
4 +DDMM
5 +DDMM.m
6 +DDMMSS
7 +DDMMSS.s
Parameters:

dec : Latitude

Declination.

digits : int (>=2)

Number of digits.
Returns:

strrepr : str

IAU format string representation of the angle.
