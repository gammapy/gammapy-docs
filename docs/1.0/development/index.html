Developer guide#

The developer documentation is a collection of notes for Gammapy developers and maintainers. If something is not covered here, have a look at the very extensive Astropy developer documentation, as well. If you want to contribute to Gammapy just use one of the contact points listed on the Gammapy webpage. Ideally you join directly the #dev channel in the Gammapy Slack.
