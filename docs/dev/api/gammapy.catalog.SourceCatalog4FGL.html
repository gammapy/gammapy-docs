SourceCatalog4FGL#

class gammapy.catalog.SourceCatalog4FGL(filename='$GAMMAPY_DATA/catalogs/fermi/gll_psc_v32.fit.gz')[source]#

Bases: SourceCatalog

Fermi-LAT 4FGL source catalog.

  • https://arxiv.org/abs/1902.10045 (DR1)

  • https://arxiv.org/abs/2005.11208 (DR2)

  • https://arxiv.org/abs/2201.11184 (DR3)

  • https://arxiv.org/abs/2307.12546 (DR4)

By default we use the file of the DR4 initial release from https://fermi.gsfc.nasa.gov/ssc/data/access/lat/14yr_catalog/

One source is represented by SourceCatalogObject4FGL.
