DataManager¶

class gammapy.data.DataManager(config=None)[source]¶

Bases: object

Data manager.

The data manager helps to locate, download, index, validate and generally manage data.

The DataManager object just stores the configuration in a config attribute and that configuration can be read from and written to YAML files.
