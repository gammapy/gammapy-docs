Background 2D.

Data format specification: BKG_2D

Parameters:
axeslist of MapAxis or MapAxes
Required axes (in the given order) are:

  • energy (reconstructed energy axis)

  • offset (field of view offset axis)

Data array.

Data unit usually s^-1 MeV^-1 sr^-1.
