motion_since_birth¶

gammapy.utils.coordinates.motion_since_birth(v, age, theta, phi)[source]¶

Compute motion of a object with given velocity, direction and age.

Parameters:

Absolute value of the velocity

Age of the source.

theta : Quantity

Angular direction of the velocity.

vz : Quantity
