0.2 (Apr 13, 2015)#

Summary#

  • Released Apr 13, 2015

  • 4 contributors (1 new)

  • 8 months of work

  • 40 pull requests

  • Requires Astropy version 1.0 or later.

  • Gammapy now uses Cython, i.e. requires a C compiler for end-users and in addition Cython for developers.
